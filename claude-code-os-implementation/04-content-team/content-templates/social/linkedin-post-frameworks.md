## Framework 1: Problem-Solution-Value (PSV)

**Best For**: Educational content, thought leadership, showcasing expertise

**Structure**:
```

[Your approach, system, or insight]

THE VALUE:
[What changes when they apply this]
[Concrete outcomes, not vague benefits]

[Optional CTA]
```

```
THE PROBLEM:
Most businesses waste 30+ minutes daily on planning.
Peak energy gone before execution even starts.

THE SOLUTION:

‚Üí Single priority identified
‚Üí Tasks tier-ranked
‚Üí Time blocks energy-aligned

THE VALUE:

```

## Writing Best Practices

### Hook Guidelines (First 1-2 Lines)
- Start with curiosity, surprise, or bold statement
- Make it scannable (short sentences)
- Create pattern interrupt
- Promise specific value

### Body Guidelines
- Use white space (line breaks between ideas)
- Mix sentence lengths (short punchy + longer explanation)
- Use symbols for bullets (‚Üí ‚úÖ ‚ùå)
- Include concrete examples
- Avoid jargon unless targeting technical audience

### CTA Guidelines
- Make it low-friction (DM, comment, visit site)
- Specific action (not vague "reach out")
- Value-focused (what they get, not what you want)
- Optional: Make CTA a question for engagement

## Platform-Specific Tips

### LinkedIn
- **Optimal Length**: 150-300 words (short-medium)
- **Line Breaks**: Every 1-2 sentences
- **Hashtags**: 3-5 relevant tags at end
- **Engagement**: Ask question or invite comment
- **Posting Times**: Tuesday-Thursday, 8-10 AM or 12-1 PM EST

### Twitter/X
- **Thread Structure**: Hook tweet + 4-7 follow-up tweets
- **First Tweet**: Must work standalone
- **Thread Flow**: One idea per tweet
- **End Tweet**: Clear CTA or summary

## Content Mixing Strategy

### 70-20-10 Rule
- **70%**: Educational/Value (teach, share insights)
- **20%**: Inspirational/Personal (stories, lessons)
- **10%**: Promotional (offers, services, CTAs)

### Weekly Posting Cadence
- **Monday**: Thought leadership / Contrarian take
- **Tuesday**: Educational / How-to
- **Wednesday**: Story / Case study
- **Thursday**: Quick tips / Listicle
- **Friday**: Engagement question / Discussion

## Engagement Tactics

### Increase Comments
- End with a question
- Create debate (respectfully)
- Ask for experiences
- Invite opinions

### Increase Saves
- Actionable frameworks
- Reference-worthy lists
- Templates or checklists
- Step-by-step guides

### Increase Shares
- Contrarian insights
- Surprising data
- Strong opinions (backed by reason)
- Relatable struggles + solutions
