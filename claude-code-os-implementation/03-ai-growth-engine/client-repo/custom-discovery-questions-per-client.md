## üìã USAGE GUIDE

### How to Use These Questions:

**BEFORE OUTREACH**:
- Review the custom questions for that client
- Select 3-5 most critical questions
- Customize based on what you already know

**DURING DISCOVERY CALL**:
- Start with open-ended questions
- Use follow-ups based on their answers
- Don't interrogate - have a conversation

**AFTER CALL**:
- Document answers in client profile
- Identify problems uncovered
- Refine value hypothesis
- Prepare next steps
