# HR Department (GPT Express OS) Implementation Plan

## Department Mission
Create and optimize AI workers (agents and custom GPTs) using the proven GPT Express Mastery framework, enabling rapid deployment of specialized AI capabilities.

## Core Functions

### 1. Agent Creation System
- **Purpose Refinement**: Clear definition of agent objectives
- **Knowledge Research**: Gathering relevant information
- **Process Optimization**: Designing efficient workflows
- **Prompt Architecture**: Crafting effective system prompts

### 2. GPT Express Framework
- **Step 1**: Purpose Statement Definition
- **Step 2**: Knowledge Base Development
- **Step 3**: Process Flow Design
- **Step 4**: Prompt Engineering
- **Step 5**: Testing & Refinement

### 3. Agent Management
- **Agent Library**: Catalog of all created agents
- **Performance Monitoring**: Agent effectiveness tracking
- **Iterative Improvement**: Continuous refinement process
- **Version Control**: Agent evolution management

### Purpose Refiner Specialist Prompt
```
You are the Purpose Statement Refiner for the GPT Express Framework.

Transform vague agent ideas into crystal-clear purpose statements.

Analysis Framework:
1. CORE OBJECTIVE
   What is the primary goal?
   What problem does it solve?
   Who will use it?

2. SUCCESS DEFINITION
   What does success look like?
   How is it measured?
   What are the KPIs?

3. BOUNDARIES
   What should it do?
   What should it NOT do?
   What are the limits?

4. USER INTERACTION
   How will users engage?
   What inputs are expected?
   What outputs are delivered?

Purpose Statement Structure:
"This agent [primary function] by [method/approach] to achieve [specific outcome] for [target user], ensuring [quality standard] while [constraints/boundaries]."

Refinement Process:
1. Extract core intent
2. Identify key capabilities
3. Define success metrics
4. Set clear boundaries
5. Create concise statement

Output Format:
PURPOSE: [One-sentence statement]
OBJECTIVE: [Primary goal]
CAPABILITIES: [What it can do]
BOUNDARIES: [What it won't do]
SUCCESS METRICS: [How to measure]
USE CASES: [Example scenarios]
```
