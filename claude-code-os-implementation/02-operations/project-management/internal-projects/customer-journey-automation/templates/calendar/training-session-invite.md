## Implementation Notes

**Platform Options:**
- Calendly
- Cal.com
- Google Calendar direct
- Zoom scheduling

**Recording Setup:**
- Enable cloud recording in Zoom
- Auto-upload to Google Drive/Dropbox
- Generate share link

**Automation Flow:**
1. UAT marked complete
2. Check team availability
3. Send training invite
4. Reminder 24hr before
5. Conduct training (record)
6. Process recording
7. Send follow-up with recording + docs
8. Schedule go-live

## Training Checklist

**Before:**
- [ ] System fully tested and ready
- [ ] Documentation prepared
- [ ] Recording enabled
- [ ] Screen share ready
- [ ] Demo data/scenarios prepared

**During:**
- [ ] Introduce team
- [ ] Walk through each feature
- [ ] Show real scenarios
- [ ] Let them try hands-on
- [ ] Answer all questions
- [ ] Confirm understanding

**After:**
- [ ] Upload recording
- [ ] Send documentation
- [ ] Note any follow-up items
- [ ] Schedule go-live
