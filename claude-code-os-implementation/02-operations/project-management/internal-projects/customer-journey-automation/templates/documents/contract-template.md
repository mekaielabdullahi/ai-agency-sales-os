# Contract Template

**Stage:** 4 - Implementation Proposal
**Touchpoint:** #19
**Type:** Document
**Automation:** Auto-generate (populate with deal details)

---

## Purpose

Service agreement for implementation projects.

---

## Document Structure

### 1. Header

```
PROFESSIONAL SERVICES AGREEMENT

Between:
AriseGroup.ai ("Provider")
[Address]

And:
[Client Company Name] ("Client")
[Client Address]

Effective Date: [Date]
```

---

### 2. Scope of Services

```
SCOPE OF WORK

Provider agrees to deliver the following services:

Project Name: [Project Name]

Deliverables:
1. [Deliverable 1]
2. [Deliverable 2]
3. [Deliverable 3]

Timeline:
- Project Start: [Date]
- Estimated Completion: [Date]

Out of Scope:
- [Item 1]
- [Item 2]
```

---

### 3. Investment & Payment Terms

```
INVESTMENT

Total Project Investment: $[Amount]

Payment Schedule:
- [X]% ($[Amount]) due upon signing
- [X]% ($[Amount]) due at [Milestone]
- [X]% ($[Amount]) due upon completion

Payment Methods:
- Bank Transfer (ACH)
- Credit Card (+3% processing fee)
- Check

Late Payment:
Invoices not paid within 30 days will incur a 1.5% monthly late fee.
```

---

### 4. Project Terms

```
PROJECT EXECUTION

Communication:
- Weekly status updates via email
- Primary contact: [Name]
- Response time: 24-48 business hours

Client Responsibilities:
- Provide access to required systems within [X] days
- Respond to requests within [X] business days
- Designate decision-maker for approvals
- Attend scheduled meetings

Change Requests:
Changes to scope require written agreement and may affect timeline and investment.
```

---

### 5. Intellectual Property

```
INTELLECTUAL PROPERTY

Work Product:
Upon full payment, Client owns all custom work created specifically for this project.

Provider Tools:
Provider retains ownership of pre-existing tools, frameworks, and methodologies.
Client receives a perpetual license to use these within their delivered solution.

Third-Party Tools:
Third-party software remains subject to their respective licenses.
```

---

### 6. Confidentiality

```
CONFIDENTIALITY

Both parties agree to keep confidential all proprietary information shared during this engagement.

This includes but is not limited to:
- Business processes and strategies
- Customer data
- Pricing and financial information
- Technical specifications

Duration: [X] years from project completion
```

---

### 7. Warranties & Limitations

```
WARRANTIES

Provider warrants:
- Services will be performed professionally
- Deliverables will function as specified
- [X] days of bug fixes post-launch included

LIMITATIONS

Provider is not liable for:
- Issues caused by third-party services
- Client modifications to delivered work
- Consequential or indirect damages

Maximum liability limited to fees paid under this agreement.
```

---

### 8. Term & Termination

```
TERMINATION

Either party may terminate with [X] days written notice.

Upon termination:
- Client pays for work completed to date
- Provider delivers all completed work
- Both parties return confidential information

Refunds:
- Before project start: Full refund minus admin fee
- After project start: Pro-rated based on work completed
```

---

### 9. Signatures

```
AGREEMENT

By signing below, both parties agree to the terms of this agreement.


PROVIDER:
AriseGroup.ai

Signature: _________________________
Name: [Name]
Title: [Title]
Date: _________________________


CLIENT:
[Client Company Name]

Signature: _________________________
Name: [Name]
Title: [Title]
Date: _________________________
```
