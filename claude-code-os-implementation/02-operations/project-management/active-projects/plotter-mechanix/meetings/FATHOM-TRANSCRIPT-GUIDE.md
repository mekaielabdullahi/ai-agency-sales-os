# Fathom Transcript Import Guide

## Quick Steps for Each Fathom Recording:

1. **Open the Fathom Share Link**
2. **Click "View Transcript"** (usually below the video)
3. **Select All Text** (Cmd+A)
4. **Copy** (Cmd+C)
5. **Paste into corresponding folder's transcript.md file**

## Transcript Template

When saving each transcript, use this format:

```markdown
# [Meeting Title]

## Metadata
- **Date**: [YYYY-MM-DD]
- **Participants**: [Names]
- **Duration**: [XX minutes]
- **Source**: Fathom/Fireflies
- **Link**: [Original URL]

## Transcript

[Paste full transcript here]
```
