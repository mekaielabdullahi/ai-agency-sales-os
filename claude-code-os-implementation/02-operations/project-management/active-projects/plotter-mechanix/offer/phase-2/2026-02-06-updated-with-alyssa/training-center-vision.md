### 2. Progressive Learning Pathways

**New Tech Journey:**

**Weeks 1-2: Foundation (Remote Learning)**
- Watch core videos
- Pass safety quiz
- Shadow via video calls
- Practice in Jobber sandbox

**Weeks 3-4: Guided Field Work**
- Ride along with senior tech
- Handle basic tasks under supervision
- Use mobile app for guidance
- Document learning in system

**Weeks 5-8: Supervised Independence**
- Solo simple calls (oil changes, cleaning)
- Kelsey available via video
- AI assistant for common questions
- Senior tech backup if needed

**Weeks 9-12: Full Independence**
- Handle 70% of calls solo
- Escalate complex to Kelsey
- Start contributing to knowledge base
- Mentor newer techs

**Months 4+: Knowledge Contributor**
- Document new solutions
- Create troubleshooting guides
- Answer forum questions
- Potentially create videos
