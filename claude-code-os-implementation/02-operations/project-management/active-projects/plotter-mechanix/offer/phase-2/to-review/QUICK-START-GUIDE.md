## üî• What Could Go Wrong (And How to Handle It)

### Problem 1: Kelsey/Alyssa Don't Track Time
**Mitigation:**
- Simplify templates (fewer fields)
- Offer to text/call daily for verbal estimates
- Track just 2-3 days instead of 5

### Problem 2: Validated ROI Much Lower Than Estimated
**Mitigation:**
- Reduce price to maintain 2.5:1 ratio
- OR reduce scope (modular approach)
- OR focus on highest-ROI deliverables only

### Problem 3: Megan/Andrew Don't Want to Collaborate
**Mitigation:**
- Reduce or remove their deliverables
- Focus on independent systems (training, Equipment CRM)
- Adjust pricing accordingly

### Problem 4: Jobber Data Export Fails
**Mitigation:**
- Work with partial data (top 50 customers)
- Manual data collection (Kelsey provides summary)
- Use conservative estimates, disclose assumptions

### Problem 5: Budget Ceiling Lower Than $47k
**Mitigation:**
- Offer Phase 2A ($23k) + Phase 2B ($24k) split
- Modular approach (choose top 3 deliverables)
- Payment plan (3-4 monthly installments)
