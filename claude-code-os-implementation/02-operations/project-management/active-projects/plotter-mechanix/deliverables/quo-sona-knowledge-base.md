## Technical Terminology Glossary

### Printer Terms
| Term | Definition |
|------|------------|
| **Plotter** | Large-format printer, often used for CAD/architectural drawings |
| **Wide-format printer** | Printers 17" or wider |
| **Large-format printer** | Printers 24" or wider |
| **PostScript** | Adobe page description language for precise graphics |
| **Aqueous ink** | Water-based ink (dye or pigment) |
| **Pigment ink** | Water-based ink with pigment particles, more durable |
| **Dye ink** | Water-based ink, vibrant colors |
| **Latex ink** | Water-based ink with latex polymers |
| **Solvent ink** | Petroleum-based ink for outdoor durability |
| **Eco-solvent ink** | Milder solvent ink, lower odor |
| **UV curable ink** | Ink cured by UV light |
| **Dye-sublimation ink** | Ink that sublimates into fabric |
| **Printhead** | Component that deposits ink onto media |
| **Nozzle** | Tiny opening in printhead that ejects ink |
| **Cartridge** | Replaceable ink container |

### Cutter Terms
| Term | Definition |
|------|------------|
| **Cutting plotter** | Machine that cuts vinyl, film, or paper |
| **Vinyl cutter** | Cutting plotter for vinyl materials |
| **Roll-fed** | Cutter that accepts media from a roll |
| **Contour cutting** | Cutting around printed graphics |
| **Die cutting** | Cutting shapes through material |
| **Through cut** | Cutting completely through material |
| **Perforated cut** | Cutting with small uncut sections (for easy tear-off) |
| **Kiss cut** | Cutting through top layer only, not backing |
| **ARMS** | Automatic Registration Mark Sensing (for print-and-cut alignment) |
| **Registration marks** | Printed marks for cutter alignment |
| **Print & Cut** | Workflow where graphics are printed then cut |

### Media Terms
| Term | Definition |
|------|------------|
| **Bond paper** | Standard uncoated paper |
| **Coated paper** | Paper with coating for better ink adhesion |
| **Resin coated (RC)** | Photo paper with resin layer |
| **Polypropylene** | Plastic film media |
| **Adhesive vinyl** | Vinyl with sticky backing |
| **Pressure-sensitive adhesive** | Adhesive activated by pressure |
| **Release liner** | Backing paper on adhesive media |
| **Window tint** | Thin film for automotive/architectural windows |
| **Reflective** | High-visibility material for signs |
| **Magnet** | Magnetic sheet material |

### Finishing Terms
| Term | Definition |
|------|------------|
| **Laminator** | Machine that applies protective film |
| **Cold lamination** | Laminating without heat |
| **Hot lamination** | Laminating with heat activation |
| **Mounting** | Adhering prints to rigid substrates |
| **Finishing** | Post-print processes (laminating, cutting, mounting) |

### Service Terms
| Term | Definition |
|------|------------|
| **Clogged printhead** | Blocked nozzles preventing ink flow |
| **Printhead recovery** | Cleaning/restoring clogged printheads |
| **Nozzle check** | Test print to verify printhead health |
| **Head cleaning** | Maintenance cycle to clear nozzles |
| **Purge** | Forcing ink through to clear blockages |

## Quick Reference - Common Customer Questions

**"What ink do I need?"**
→ Ask for printer model, then reference HP ink number chart above

**"My prints have lines/streaks"**
→ Likely clogged printhead - may need cleaning or recovery service

**"How much does service cost?"**
→ Depends on issue - schedule diagnostic call

**"Do you sell [brand] printers?"**
→ Yes, HP and Graphtec new equipment; refurbished units for other brands

**"What paper should I use?"**
→ Depends on application:
- CAD/blueprints → Bond paper
- Photos → Photo paper (satin or gloss)
- Signage → Coated paper or poster paper
- Stickers → Adhesive vinyl or polypropylene

**"Do you service my area?"**
→ Arizona-based, primarily Phoenix metro area
