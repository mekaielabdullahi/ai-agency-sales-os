# Daily Planner Agent Prompt

This file contains the core prompt for the daily planning agent.

See the full agent specification in:
`../../agents/daily-planner-agent.md`

## Quick Reference

**Purpose**: Generate focused, prioritized daily roadmaps in under 60 seconds

**Key Outputs**:
- THE ONE THING
- Tier-based task list (T1, T2, T3)
- Energy management strategy
- Kill list
- Success metrics

**Usage**: Run every morning before starting work
