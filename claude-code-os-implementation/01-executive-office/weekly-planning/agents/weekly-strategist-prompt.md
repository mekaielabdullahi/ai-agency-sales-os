## Quick Reference

**Purpose**: Generate strategic week plans with day-by-day execution roadmaps

**Key Outputs**:
- Week's ONE THING
- Daily execution plans
- Kill list
- Energy management strategy
- Risk mitigation plan

**Usage**: Run every Sunday evening for next week planning
