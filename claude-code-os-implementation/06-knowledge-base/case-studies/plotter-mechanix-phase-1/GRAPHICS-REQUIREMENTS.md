## Brand Guidelines

### Colors

| Color | Hex Code | RGB | Usage |
|-------|----------|-----|-------|
| Navy | #1E3A5F | 30, 58, 95 | Primary backgrounds, text |
| Amber | #F59E0B | 245, 158, 11 | Accent highlights, CTAs |
| Gold | #D4AF37 | 212, 175, 55 | Premium accents |
| Teal | #0D9488 | 13, 148, 136 | Icons, secondary accents |
| White | #FFFFFF | 255, 255, 255 | Text on dark backgrounds |
| Dark Gray | #374151 | 55, 65, 81 | Body text |

### Typography

| Use | Font Family | Weight | Notes |
|-----|-------------|--------|-------|
| Headlines | Inter or similar sans-serif | Bold (700) | Large, impactful |
| Body | Inter or similar sans-serif | Regular (400) | Clean, readable |
| Numbers/Stats | Inter or similar | Black (900) | Extra emphasis |
| Quotes | Georgia or similar serif | Italic | Personal feel |

## Animation Specifications

### General Guidelines

| Property | Value |
|----------|-------|
| Transitions | Smooth, 0.5-1s duration |
| Easing | Ease-out for emphasis |
| Style | Professional, not flashy |

### Number Counters

| Property | Value |
|----------|-------|
| Duration | 2-3 seconds |
| Easing | Ease-out for dramatic effect |
| Emphasis | Brief scale-up on final number |

### Diagram Builds

| Property | Value |
|----------|-------|
| Reveal | Step-by-step |
| Arrows | Animate in sequence |
| Icons | Fade + scale in |
| Timing | 0.3-0.5s per element |
