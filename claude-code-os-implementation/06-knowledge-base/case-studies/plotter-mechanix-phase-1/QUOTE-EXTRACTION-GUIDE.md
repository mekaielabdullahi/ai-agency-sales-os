## Quote Formatting for Video

### On-Screen Text
- Use sentence case
- Max 2 lines preferred
- Include speaker name below quote
- Consider branded quote card design

### Example Treatment
```
"I'm starting to trust it."
         — Kelsey, Owner
```

### Voiceover Quotes
When using quotes as VO, ensure:
- Audio is clean
- Background noise is minimized
- Quote is complete (not cut off)
- Natural inflection preserved
