# Processed YouTube Insights

## How to Use

This folder holds **structured insights** extracted from raw transcripts.

### When to Create a File Here

- You've processed a raw transcript
- You found actionable tactics worth testing
- The insight aligns with your agency objectives

### File Template

```markdown
# [Video Title] - Key Insights

**Source**: [Channel Name]
**URL**: [YouTube URL]
**Date Processed**: [Date]
**Relevance**: [Sales / Delivery / Content / Operations / Pricing / Mindset]

---

## Key Insights

### 1. [Insight Title]

**The Tactic**:
[What they do/recommend]

**Why It Works**:
[Their reasoning]

**How to Apply at AriseGroup**:
[Your specific application]

**Status**: [ ] To Test  [ ] Testing  [ ] Validated  [ ] Rejected

---

## Action Items

- [ ] [Specific action to take]
- [ ] [Update to make to playbook/OS]

---

## If Validated

Promote to: `/agency-playbooks/[relevant-playbook].md`
```

### Naming Convention

`YYYY-MM-DD-topic-or-tactic.md`
- Example: `2026-01-18-pricing-value-based-framework.md`

---

*Only keep insights you'll actually test. Delete the rest.*
