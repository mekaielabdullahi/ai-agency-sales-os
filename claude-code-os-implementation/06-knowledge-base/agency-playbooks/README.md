# Agency Playbooks

**Purpose**: Living operational playbooks that define how AriseGroup.ai executes. These are the "how we do things here" documents that your team follows.

---

## How This Differs From Other Folders

| Folder | Purpose |
|--------|---------|
| `/frameworks/` | Mental models for thinking |
| `/methodologies/` | Step-by-step approaches |
| `/best-practices/` | Tactical tips |
| **`/agency-playbooks/`** | **Complete operational systems your team follows** |

Playbooks are end-to-end. They combine frameworks + methodologies + best practices into executable workflows.

## Playbook Template

Every playbook follows this structure:

```markdown
# [Playbook Name]

## Purpose
What this playbook achieves.

## When to Use
Trigger conditions for using this playbook.

## Prerequisites
What needs to be true before starting.

## The Playbook

### Phase 1: [Name]
**Goal**: What this phase achieves
**Time**: Expected duration
**Steps**:
1. Step one
2. Step two
3. Step three

**Output**: What you have when done

### Phase 2: [Name]
[Continue pattern...]

## Quality Checklist
- [ ] Check 1
- [ ] Check 2
- [ ] Check 3

## Common Mistakes
- Mistake 1: How to avoid
- Mistake 2: How to avoid

## Version History
- v1.0 (Date): Initial version
- v1.1 (Date): Updates based on [learning]
```

---

## How to Update Playbooks

1. **After every execution**: Note what worked/didn't
2. **Weekly**: Review notes, update playbooks
3. **Monthly**: Major revisions based on patterns
4. **Version everything**: Track changes over time
