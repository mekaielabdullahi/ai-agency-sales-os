# Slack Best Practices

## Channel Prefixes

| Prefix | Category |
|--------|----------|
| `all-` | Company-wide |
| `client-` | Client acquisition |
| `dev-` | Development |
| `mktg-` | Marketing |
| `ops-` | Operations |
| `sales-` | Sales & revenue |

## Message Routing

1. Specific client? → client channel
2. Legal/contracts? → #ops-agreements
3. New lead/sales? → #sales-pipeline
4. Technical work? → #dev-main
5. Prompt engineering? → #dev-prompt
6. Link/resource? → #ops-resources
7. Content/branding? → #mktg-general
8. Casual chat? → #ops-general
