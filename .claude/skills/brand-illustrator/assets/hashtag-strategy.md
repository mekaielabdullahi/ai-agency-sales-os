# AriseGroup.ai Hashtag Strategy

## Overview

Hashtags on LinkedIn serve two purposes: discovery and positioning. We use them strategically, not excessively. Quality over quantity - 3-5 targeted hashtags per post.

---

## Hashtag Tiers

### Tier 1: Brand Hashtag (Always Use)
```

```

- Use on every post

## Hashtag Formulas by Post Type

### Insight Posts
```
#AriseWithAI #AIforBusiness + [1 topic hashtag]
```
Example: `#AriseWithAI #AIforBusiness #BusinessEfficiency`

### Client Story Posts
```
#AriseWithAI + [result hashtag] + [industry if relevant]
```
Example: `#AriseWithAI #BusinessGrowth #SmallBusinessGrowth`

### How-To Posts
```
#AriseWithAI + [topic hashtag] + [action hashtag]
```
Example: `#AriseWithAI #BusinessAutomation #ProcessImprovement`

### Engagement Posts
```
#AriseWithAI + [broad topic] + [community hashtag]

Example: `#AriseWithAI #AIforBusiness #FutureOfWork`

### Announcement Posts
```
#AriseWithAI + [service hashtag] + [audience hashtag]

## Hashtags to Avoid

### Too Broad (Low Value)
- #Business
- #Technology
- #Success
- #Entrepreneur
- #Motivation

### Too Trendy/Spammy
- #AI (too generic, too competitive)
- #ChatGPT (unless specifically about it)
- #Innovation (overused)
- #Disruption (against brand voice)
- #Hustle (wrong vibe)

### Engagement Bait
- #FollowForFollow
- #Like4Like
- #Comment
- #Viral

---

## Placement Guidelines

### Where to Put Hashtags
**Option A: End of post (preferred)**
```
[Post content]

What would you automate first?

#AriseWithAI #AIforBusiness #BusinessAutomation
```

**Option B: First comment**
Post hashtags as first comment to keep post clean. Works well for longer posts.

### Formatting
- All hashtags on one line or grouped naturally
- No hashtags mid-sentence (breaks reading flow)
- No hashtag-stuffing (max 5)
