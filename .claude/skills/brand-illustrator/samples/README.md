# Sample Illustrations

Add approved brand illustrations here as reference for style consistency.

## Purpose

These samples help:
1. Train AI generation to match your established style
2. Provide visual reference during concept development
3. Ensure consistency across all generated illustrations

## Recommended Samples

Add 5-10 illustrations that represent:
- [x] Different topic categories (audit, automation, strategy, etc.)
- [x] Various formats (square, landscape, hero)
- [x] Range of compositions (focus, system, transformation)
- [x] Approved color usage examples

## Naming Convention

```
[topic]-[format]-[version].png

Examples:
- ai-audit-linkedin-v1.png
- automation-blog-hero-v2.png
- strategy-presentation-v1.png
```

## Adding New Samples

When you generate an illustration you're happy with:

1. Copy from project folder to this samples folder
2. Rename following the convention above
3. These will be referenced for future generation consistency
